# tinymce-chatgpt-plugin
A TinyMCE plugin for using ChatGPT

-- Ref

In Nova4 TinyMCE Nova Package

```php
'openai' => [
            'api_key' => 'sk-yourapikeyhere',
            'model' => 'text-davinci-003',
            'temperature' => 0.5,
            'max_tokens' => 150,
            'prompts' => [
                'Translate from English to Italian',
                'Summarize',
                'Proofread',
                'Write a blog post about',
            ]
        ],
```
